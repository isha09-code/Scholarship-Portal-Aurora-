# 🌟 Aurora Scholarship Portal

💡 *Empowering students by making scholarships accessible, simple, and trackable online.*

---

## 📌 Problem Statement:

**Problem Statement 12 – Redesign Crisis Logistics and Relief Distribution**

---

## 🎯 Objective

The **Aurora Scholarship Portal** addresses the challenge of **scattered and hard-to-find scholarship opportunities**, especially for rural students.

✨ It provides a **centralized, responsive, and easy-to-use platform** where users can:

* View scholarships
* Apply seamlessly
* Track application status

🎯 My goal is to **reduce barriers** and **increase access to education funding for better outcome**.

---

### 🚀 Our Approach

* Focused on **accessibility & simplicity**
* Built using **pure frontend technologies**
* Ensured **responsive UI for all devices**

### ⚡ Key Highlights

* 📊 Integrated **Google Charts** for analytics
* 💾 Used **LocalStorage** for data persistence
* 🎨 Designed a clean, modern UI

---

## ✨ Key Features

✅ Easy scholarship browsing & application
✅ 📊 Dynamic charts (Google Charts)
✅ 📱 Fully responsive design
✅ 💾 LocalStorage-based data saving
✅ 👨‍🎓 Student & Admin interfaces
✅ 📄 Application tracking system
